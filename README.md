<p align="center"><a href="https://docs.nestjs.cn/" target="_blank">Nest</a> 是一款用于构建高效且可伸缩的服务端应用程序的渐进式 <a href="http://nodejs.org" target="_blank">Node.js</a> 框架</p>

## 安装

```bash
$ npm install
```

## 启动

```bash

# 开发环境
$ npm run start:dev

# 生产环境
$ npm run start:prod
```
